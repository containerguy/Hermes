       - Mock `web-push` so `sendNotification` can be forced to fail.
       - Assert that after 3 consecutive failures, the subscription is revoked (revokedAt set) and does not prevent other subscriptions from being attempted.
       - Assert that a 410/404 revokes immediately.
       The test must run without real network calls.
